# COVID-19 Tracker
This is an applicaiton that allows users to track their COVID symptoms. 

## Tech used:
MongoDB, Node.js, EJS, Passport, Javascript, CSS, HTML

## Installation
1. Clone repo
2. run `npm install`

## Usage
1. run `node server.js`
2. Navigate to `localhost:2000`
